Séparation entre contenus et présentation

Un contenu peut être présenté sous différentes formes : HTML, XML, JSON ...

La présentation peut être amenée à évoluer, le contenu d'un document ne doit pas être impacté

=> éviter les mises en forme non sémantiques dans le langage de présentation : <br/><br/><br/><br/><br/><br/>

=> éviter les informations de style dans les contenus
(cf. Méta-données)
